### Layer tokenomics deck

This presentation is the internal (~20 min) deck for Tellor Layer tokenomics + operator economics.

### Build
- Open `main.tex`
- Build with LaTeX Workshop or `latexmk -pdf main.tex`

### Theme
- This project vendors the Tellor theme locally (v2 split files):
  - `Tellor.cls`, `TellorLight.cls`
  - `TellorBase.sty`, `TellorDark.sty`, `TellorLightTheme.sty`

### Assets
This deck currently references Tellor brand assets from:
- `../rewards-presentation/assets/brand/`

If you want the deck to be fully portable later, copy the required images into:
- `assets/brand/`
