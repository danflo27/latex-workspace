### Rewards presentation

This is a Tellor-branded Beamer deck using a **vendored** theme file (`Tellor.cls`) pinned to a specific version.

### Build

- Open `main.tex`
- Save to auto-build (PDF output is configured by workspace `.vscode/settings.json`)
- Use “LaTeX Workshop: View LaTeX PDF file” to preview

### Theme

- This project vendors `Tellor.cls` locally to avoid unintentional styling changes when the shared theme evolves.
- The canonical copy for this version lives in `themes/Tellor/v1/Tellor.cls`.
