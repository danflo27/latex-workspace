### Tellor theme — v1

This folder is the **canonical source** for the current Tellor Beamer theme used by
`presentations/rewards-presentation/`.

Policy:
- Presentations should **vendor/pin** these files into their project folder (copy them in).
- Avoid editing a deck’s vendored theme unless you intend to restyle that deck.

Files:
- `Tellor.cls`: dark variant
- `TellorLight.cls`: light variant
